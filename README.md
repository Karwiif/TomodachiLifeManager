# TomodachiLifeManager

Web tool for managing and organizing residents in Tomodachi Life.
Herramienta web para la gestión y organización de residentes en Tomodachi Life.

---

## English

This is a static web tool designed to plan and organize your island residents. It allows you to manage multiple islands, classify Miis by categories, and track population limits.

### Key Features

- **Population Limit:** Maximum capacity of 70 residents, matching the original game's constraints.
- **Multi-Island System:** Create and save different island configurations within the browser's memory.
- **Categorization:** Organize Miis by series, franchises, or custom groups.
- **Progress Bar:** A dynamic visual indicator that changes color based on island occupancy.
- **Data Persistence:** Automatic saving to the browser's local storage (LocalStorage).
- **Import/Export:** Support for JSON files to create backups or share data with other users.

### User Guide

1. **Installation:** Download the repository and open `index.html` in any modern browser. No internet connection required.
2. **Management:** Create new islands or import existing JSON files from the main menu.
3. **Editing:** You can rename or delete islands directly from the start screen.
4. **Backup:** Always use the **Export JSON** button to generate a physical file of your data, as browser memory may be cleared if you wipe your history.

## Castellano

Esta es una herramienta web estática diseñada para planificar y organizar los residentes de tu isla. Permite gestionar múltiples islas, clasificar Miis por categorías y visualizar el límite de ocupación.

### Características principales

- **Límite de población:** Capacidad máxima de 70 residentes, igual que en el juego original.
- **Sistema Multi-isla:** Permite crear y guardar diferentes configuraciones de islas en la memoria del navegador.
- **Categorización:** Organización de Miis por series, franquicias o grupos personalizados.
- **Barra de progreso:** Indicador visual dinámico que cambia de color según el llenado de la isla.
- **Persistencia de datos:** Guardado automático en el almacenamiento local (LocalStorage) del navegador.
- **Importación/Exportación:** Compatibilidad con archivos JSON para copias de seguridad o para compartir datos.

### Guía de uso

1. **Instalación:** Descarga el repositorio y abre `index.html` en cualquier navegador moderno. No requiere internet.
2. **Gestión:** Crea nuevas islas o importa archivos JSON existentes desde el menú principal.
3. **Edición:** Puedes renombrar o borrar islas directamente desde la pantalla de inicio.
4. **Respaldo:** Usa siempre el botón **Exportar JSON** para generar un archivo físico de tus datos, ya que la memoria del navegador puede borrarse si limpias el historial.
